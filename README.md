#**Example of README.md**
Tα βήματα που ακολουθούμε για τη δημιουργία ενός αποθετηρίου στο Github είναι τα παρακάτω:
***Εγγραφή στο Github**
Το πρώτο βήμα για την χρήση του Github είναι η εγγραφή σε αυτό. Η διαδικασία απαιτεί τη χρήση ενός λογαριασμού ηλεκτρονικού ταχυδρομείου.
Για να εγγραφείτε αρκεί να μεταβείτε στην ιστοσελίδα {https://github.com/}.
Εκεί πατάτε στο εικονίδιο που γράφει *“Sign up”*.
Το Github θα σας ζητήσει ένα όνομα χρήστη, ένα password και μια διεύθυνση email. Στη διεύθυνση που έχετε καταχωρήσει, θα αποσταλεί ένα μήνυμα με ένα κωδικό **GitHub launch code**.Σε περίπτωση που δεν εντοπίσετε το mail, θα πρέπει να ελέγξετε την ανεπιθύμητη αλληλογραφία σας. 
Μόλις πατήσετε το σύνδεσμο, θα έχετε επιβεβαιώσει την διεύθυνση ηλεκτρονικού ταχυδρομείου και θα έχετε ολοκληρώσει επιτυχώς την εγγραφή σας. 
Υπάρχει το ενδεχόμενο το Github να σας κάνει κάποιες ερωτήσεις σχετικά με το είδος της χρήσης που θα κάνετε.
Είστε ελεύθεροι να επιλέξετε ότι σας ταιριάζει καλύτερα. Σε περίπτωση που ερωτηθείτε αν θέλετε να γραφτείτε συνδρομητές έναντι κάποιου ποσού ή όχι,
προτείνουμε να επιλέξετε το δωρεάν πλάνο.
***Δημιουργία αποθετηρίου**
Αφού έχετε ολοκληρώσει την εγγραφή σας, είστε έτοιμοι να δημιουργήσετε το πρώτο σας αποθετήριο.
Αυτό γίνεται πατώντας στο κουμπί *“Create repository”*. Θα σας ζητηθεί να δώσετε κάποιο όνομα για το αποθετήριο. 
Προαιρετικά μπορείτε να δώσετε μια περιγραφή. Μπορείτε επίσης να επιλέξετε αν το αποθετήριο θα είναι δημόσιο ή ιδιωτικό.   
Τέλος, μαρκάρετε την επιλογή *“Add a README file”* και πατήστε το κουμπί *“Create repository”* . 
**Συγχαρητήρια! Μόλις δημιουργήσατε το πρώτο σας αποθετήριο.**
